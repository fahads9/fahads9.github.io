Delegation Chain — Field-Level Bindings

Each layer cryptographically binds the next via cnf.jwk (RFC 7800). Selective disclosure per RFC 9901

Always visible
Selectively disclosable
REQ Required
OPT Optional
REC Recommended
cnf.jwk → signs next layer
sd_hash = hash(prior layer)
kid ↔ cnf.kid match

Signature & Binding Summary · Autonomous (L1 + L2 + L3)

L1 signed by Issuer · cnf.jwk = User Key · typ: sd+jwt · ~1yr
→
L2 signed by User (L1 cnf) · sd_hash→L1 · mandate cnf.jwk = Agent Key · typ: kb-sd-jwt+kb · 24h–30d
→
L3 signed by Agent (L2 cnf) · sd_hash→L2+discl · header kid = L2 cnf.kid · typ: kb-sd-jwt · ~5min
L2↔ payment.reference: conditional_transaction_id = hash(checkout disclosure) — pre-commits payment → checkout at mandate creation

Signature & Binding Summary · Immediate (L1 + L2)

L1 signed by Issuer · cnf.jwk = User Key · typ: sd+jwt · ~1yr
→
L2 signed by User (L1 cnf) · sd_hash→L1 · no cnf — terminal · typ: kb-sd-jwt · ~15min

L3b — Checkout
